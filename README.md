# SportBook

Aplicació web de reserves de pistes esportives (Laboratori de Projectes de Sotfware - Convocatòria Extraordinària, curs 2025-2026).

## Continguts del repositori

```
sportbook/
├── README.md               # Aquest fitxer
├── doc/
│   └── REPORT.md           # Memòria tècnica del projecte
└── src/                    # Tot el codi font (Next.js full-stack)
    └── README.md           # Instruccions per compilar, executar i provar el producte
```

## Tecnologies

| Capa | Tecnologia |
|------|-----------|
| Frontend | Next.js 14, TypeScript, Tailwind CSS |
| Backend | Next.js API Routes, TypeScript |
| Base de dades | PostgreSQL via Supabase |
| Autenticació | Supabase Auth |
| Desplegament | Vercel |
